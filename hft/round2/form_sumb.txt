Постараюсь уложиться в 1-2 предложения для каждой фичи, иначе будет не читаемо:
Accumulation Distribution line - технический индикатор, используемый для измерения давления покупок и продаж
Average  Directional Index - технический индикатор, используемый для измерения силы тренда
Change Momentum oscillator -  технический индикатор, используемый для измерения скорости изменения цены ценной бумаги
Momentum - аналогично предыдущему только этот в абсолютных значения
Rate of Change - это технический индикатор, используемый для измерения процентного изменения цены ценной бумаги за определенный период времени (тоже супер похож на предыдущий)
Stochastic Relative Strength Index - технический индикатор, используемый для измерения силы цены ценной бумаги (RSI) по отношению к ее недавним показателям (стохастически)
Linear regression line - берем веса из статистический инструмент, используемого для анализа взаимосвязи между двумя переменными.
Autocorrelation - статистический инструмент, используемый для измерения корреляции (зависимости) между ценой и ценой с некоторой задержкой
Partial Correlation - Похож на предыдущий, но под контролем влияния других переменных
Cointegration - статистический инструмент, используемый для анализа долгосрочной связи между двумя нестационарными временными рядами
Order book imbalance - это мера разницы между количеством заявок на покупку и продажу при заданном уровне цен на рынке 
Adaptive Logistic Regression - фича, которую предложили авторы. Используем LogReg, чтобы оценить тренд.
Realized Volatility - мера фактической волатильности ценной бумаги за определенный период времени
Realized Kernel - мера волатильности ценной бумаги на основе высокочастотных данных. Более сглаженная в сравнении с предыдущим
Jump Variation -  это мера размера и частоты резких изменений цены ценной бумаги (модификация Realized Volatility)
